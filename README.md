📌 Smart Complaint Management System

A web-based complaint management system designed for hostels and small residential societies. This system replaces traditional manual complaint registers with a digital, real-time, and efficient solution.

🚀 Overview

The Smart Complaint Management System allows students to submit and track complaints, while enabling the admin (warden) to manage, resolve, and monitor issues efficiently.

It ensures:

Transparency

Accountability

Real-time updates

Easy access from any device

✨ Features

👨‍🎓 Student Module

Secure login using room number and password

Submit complaints with description and optional image

Track complaint status (Pending / Resolved / Rejected)

View complaint history

👨‍💼 Admin Module

Secure admin login

View all complaints in real-time

Resolve complaints with proof image

Reject complaints with reason

Manage student records

Reset student passwords

🛠️ Tech Stack

Frontend: HTML5, CSS3, Vanilla JavaScript

Backend: Firebase (Firestore + Storage)

Hosting: Netlify / GitHub Pages

⚡ Key Highlights

Real-time updates using Firebase onSnapshot()

Cloud-based data storage

No backend server required

Lightweight and easy to deploy

Beginner-friendly architecture

📊 Problem Solved

Traditional complaint systems suffer from:

Lost or untracked complaints

No transparency

Lack of accountability

Limited accessibility

This system provides a digital and structured solution to overcome these issues.

🔐 Authentication

Student login via room number + password

Admin login via configured credentials

Session-based authentication using browser storage
